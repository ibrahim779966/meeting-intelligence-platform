# Meeting Intelligence Platform

AI-powered tool that analyzes meetings and generates summaries.

## Quick Start
1. Run backend: `cd backend && pip install -r requirements.txt && python main.py`
2. Run frontend: `cd frontend && npm install && npm start`  
3. Open http://localhost:3000

## Features
- Upload meeting recordings
- AI-powered transcripts & summaries
- Action item extraction
- Search across all meetings
- Shareable insights
